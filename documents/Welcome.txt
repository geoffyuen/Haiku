###Haiku is a flat-file note taking web app

Create notes in Markdown markup, hit save and it appears in the navigation to the left. And because it uses markdown you can format your notes using anything that a web page can use and also embed regular html, css and javascript.

Haiku was built using PHP, Markdown and Twitter Bootstrap.

###Getting Started

Click a link to the left to get information or New+ to add 
a new item in the sidebar. See [Help](?p=Help) to learn how to format your text.
